== Description ==
* BBS e-Commerce는 BBS e-theme에서 제작한 쇼핑몰 플러그인 입니다. <br />
회원관리, 상점관리, 상품관리, 상품별 SEO, 주문관리, 한국형결제시스템, 본인인증, 이용후기, 적립금, 메일링, SMS 등 국내 쇼핑몰 환경에 맞춘 풍부한 기능이 포함되어 있습니다.
